# SEACO360

# Client

Frontend for SEACO360 visualisation platform

## Installation

### Prerequisites

Install `typescript` globally

```bash
npm install -g typescript
```

When cloning the repo first time:

```bash
npm i # installs all dependent packages under node_modules
```

#### To go with the latest version please copy and past in your terminal the following steps

```
git clone https://github.com/chuleeshen/SEACO360.git && cd SEACO360/Client
```

Development mode

```
npm install && npm start
```

# Server

Backend for SEACO360 visualisation platform

Development mode

```
npm install && npm start
```
